简介

主菜单的扩展分为两部分：

  1. Dolag子菜单，在菜单栏中添加名为Dolag的菜单项。
  2. 修改现有菜单，即在原本就有的菜单选项中增添选项。

菜单

Dolag子菜单

有如下功能：

  • Visit Github。访问github。
  • Edit Configure Json。编辑DolagPlugin.json，建议不要修改，除非想加入新的环境变量。
  • Open Plugin Folder。打开HOUDINI_USER_PREF_DIR/DolagPlugin/。
  • Open Doc。在Houdini内置浏览器打开Dolag Houdini Toolset的文档。
  • Open Doc Online。使用系统默认浏览器打开Dolag Houdini Toolset的文档。
  • Install Pip。为当前Houdini的Python环境安装pip。
  • Add Tuna Mirror。添加清华源。
  • Pip Install。使用pip安装或卸载指定版本的模块。
  • Clear Python Cache。清除所有.pyc文件和pycache文件夹，主要用来强制进行python文件的热重载。

对现有菜单的修改

File菜单

  • Save Incrementally
    增加了增量保存的选项，会在$HIP目录下复制此工程并对工程名进行编号或增加编号，以增量保存。 增量保存后不会切换目前工程到保存后的工程。
  • Save Backup
    备份工程，和Houdini自带的备份功能一致。
