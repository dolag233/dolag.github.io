简介

当鼠标在变量上右键时会唤出右键菜单，此工具集的所有变量菜单扩展功能都在此右键菜单的Dolag分类中，只有在变量类型和该功能的类型相符时才会显示该功能。

功能

Copy& Paste Button

对按钮类型的变量进行复制和粘贴。 在复制了一个按钮变量后，在其他的按钮变量上复制，则在点击原按钮后，其他复制到的按钮都会产生一个点击事件。

Set Ramp Interpolation

能够将Ramp变量的所有位置都设置为同一个插值类型。

Mirror Ramp

将Ramp左右镜像 有两种模式：

  • From Left To Right：从Ramp的中间分开，将中间左边的所有值复制给右边的值。
  • From Right To Left：从Ramp的中间分开，将中间右边的所有值复制给左边的值。

Edit In VSCode

仅对Python和Vex的代码变量有效，点击后会使用VSCode打开已有的代码，但是在VSCode中编辑的代码不会同步到Houdini。

Save Snippet

仅对Python和Vex的代码变量有效，点击后会将代码片段保存到HOUDINI_USER_PREF_DIR/DolagPlugin/python/custom和HOUDINI_USER_PREF_DIR/DolagPlugin/vex/custom中。在编写Vex代码时需要符合Vex中的函数定义规则。

Import Snippet

仅对Python和Vex的代码变量有效，点击后会分别列出处在搜索路径中的所有Python方法或Vex函数，并支持按下回车进行自动include，用来看哪些头文件里有哪些函数，蛮方便的。
