Houdini AI助手

简介
...

使用openai的接口实现的AI助手，能够帮你找到你想要的节点、帮你解决代码问题、其他Houdini相关问题。注意需要用户自己提供OpenAI API Key。

打开窗口
...

目前有两种方式打开Houdini Master窗口。

  • 在主菜单中打开。主菜单 > Dolag > Houdini Master AI。
  • 在快速命令中打开。调出快速命令界面后，搜索Ask Houdini Master或chat。

配置
...

开始使用前需要进行配置。如果没有配置，则会显示一行红色的警告文字。
按下右下角的Set按钮打开如下的配置界面：

  • Openai API Key。OpenAI的API Key，需要自行申请或购买。
  • GPT Version。ChatGPT的版本，可以到此网站查看所有版本及其收费标准。目前建议的版本是gpt-4-1106-preview。
  • API URL。提供API服务的URL，如果是官网购买的key则使用https://api.openai.com/v1，如果是在其他渠道购买的API Key，则输入对应渠道的API服务的URL，例如https://oneapi.xty.app/v1。
  • HTTP Proxy & HTTPS Proxy。设置Houdini内全局的http和https代理。如果您的IP所在地区不支持OpenAI的服务则需要通过魔法开启http&https代理服务，并填写对应的网址和端口。
    设置完成后点击Save即可保存设置。

使用方法
...

有四种对话模式，可以随时切换：

  • Node。用户输入想要实现的功能，AI帮你搜索应该使用何种节点。
  • Vex。用Vex代码实现用户输入功能。
  • Python。用Python代码实现用户输入功能。
  • Free。AI会解答任何用户输入的Houdini相关问题。
    在下方的输入框输入提示词，而后按回车或点击Send按钮即可进行对话。
    按下Reset按钮可以重置当前对话。
