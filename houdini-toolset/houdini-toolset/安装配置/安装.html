安装

此处为github仓库链接。在使用前需要进行安装配置：

  1. 在HOUDINI_USER_PREF_DIR文件夹下(一般来说在windows下就是C:\Users\用户名\Documents\houdini X.Y)，使用git bash运行如下命令
git clone https://github.com/dolag233/Doalg-Houdini-Toolset.git -b dev DolagPlugin
  1. 将其中的DolagPlugin.json文件拷贝到HOUDINI_USER_PREF_DIR/packages文件夹下，若没有此文件夹则进行创建后再拷贝
    这样就完成了，dev分支下一般是最新进度，而master分支会在有比较大的更新后一并合并，因此会有滞后性。
